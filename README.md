# Mission Control

Central dashboard for monitoring directory site projects, investments, scheduled AI tasks, and work activity.

## Quick Start
```bash
npm install
npm run dev
```

Then open http://localhost:3333

## Tech Stack
- React + Tailwind CSS (frontend)
- Node.js + Express (backend)
- SQLite (database)
- Vite (build tool)
